用户须知

PaperShield 用于你有权编辑的学术草稿，帮助提升论证清晰度、表达自然度、术语一致性和引文可核查性，不承诺规避或击败任何外部检测系统。

  • 所有事实、论点、术语、数据和引注必须由你人工复核。
  • 页面中的指标是本地代理信号，不等同于任何检测器结果。
  • 配置外部模型后，草稿文本和模型调用信息可能发送至你选择的服务商，请自行确认服务商条款并妥善保管 API key。
